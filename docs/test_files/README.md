# Test Files

This folder contains C-minus test programs used to validate compiler features.

## Index

- `teste.txt`: Selection sort using a global array and array parameters.
- `teste2.txt`: Simple global array reads/writes and arithmetic.
- `teste3.txt`: Recursive GCD (Euclid) with integer returns.
- `teste4.txt`: Recursive halving with output at each step.
- `teste5.txt`: Recursive countdown (void function).
- `teste6.txt`: Recursive factorial with integer returns.
- `teste7.txt`: Recursive Fibonacci (deeper call tree).
- `testing.txt`: Human-readable checklist of scenarios (not C-minus code).

## How to run

Use `make run TEST=N` where `N` matches the test number.

Example:

```
make run TEST=2
```
